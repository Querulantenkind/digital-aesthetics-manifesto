# ASCII Art Collection

Welcome to the ASCII Art Gallery—a curated collection of text-based visual art. All works are displayed in their original form, viewable in any text editor or terminal with a monospace font.

## ASCII Art Techniques

### Character Density
Using different characters for shading:

**Light to Heavy:**
```
` . - : ; ! i l I o O @ # M W
```

Example gradient:
```
................
::::::::::::::::
;;;;;;;;;;;;;;;;
iiiiiiiiiiiiiiii
OOOOOOOOOOOOOOOO
@@@@@@@@@@@@@@@@
################
MMMMMMMMMMMMMMMM
WWWWWWWWWWWWWWWW
```

### Line-Drawing
Using characters with strong directional qualities:

```
| / \ - _ = + ~ ^ v < >
```

Example:
```
     ^
    / \
   /   \
  /     \
 /       \
/_________\
```

### Box-Drawing (Unicode)
```
┌─┬─┐  ╔═╦═╗  ╭─┬─╮
│ │ │  ║ ║ ║  │ │ │
├─┼─┤  ╠═╬═╣  ├─┼─┤
│ │ │  ║ ║ ║  │ │ │
└─┴─┘  ╚═╩═╝  ╰─┴─╯
```

### Emoticons and Kaomoji

**Western emoticons:**
```
:)  :(  ;)  :D  :P  :'(  >:(  
```

**Japanese kaomoji:**
```
(^_^)  (T_T)  (≧◡≦)  (╯°□°)╯︵ ┻━┻  (づ｡◕‿‿◕｡)づ
```

## How to View ASCII Art

1. **Use a monospace font**
   - Courier, Courier New
   - Source Code Pro, Fira Code
   - JetBrains Mono, Iosevka
   - Terminal default fonts

2. **Ensure proper character encoding**
   - ASCII art: standard ASCII (7-bit)
   - Unicode art: UTF-8 encoding

3. **Adjust viewport**
   - Some art requires wide terminals (100+ columns)
   - Zoom out if art appears broken

4. **View in plain text**
   - Text editors (Vim, Emacs, VSCode, Notepad)
   - Terminal (cat, less, more)
   - NOT in word processors (formatting breaks alignment)

## Tools for Creating ASCII Art

### Manual Tools
- Any text editor with monospace font
- Graph paper for planning
- Character reference charts

### Software
- **ASCII Art Studio** (Windows)
- **JavE** (Java-based, cross-platform)
- **Monodraw** (macOS)
- **jp2a** - JPEG to ASCII converter
- **img2txt** (libcaca) - Image to text

- ASCII Art Generator: https://www.ascii-art-generator.org/
- Text Image: https://www.text-image.com/
- TAAG (Text to ASCII Art Generator): http://patorjk.com/software/taag/

## Notable ASCII Artists

### Joan Stark (jgs)
One of the most prolific ASCII artists. Created thousands of works in the 1990s-2000s.  
Signature: `jgs` or `Joan G. Stark`

### Hayley Jane Wakenshaw (hjw)
Known for complex, detailed works. Particularly famous for oceanic themes.

### Christopher Johnson
Early ASCII artist (1960s-1970s), worked on teletypes and early computer terminals.

### Anonymous / Internet Collective
Many of the most iconic ASCII works are anonymous, created and refined collectively.

## Cultural Significance

ASCII art represents:
- **Constraint as creativity** - Limited characters, unlimited expression
- **Universality** - Renders on any device, any time
- **Accessibility** - No images required, works with screen readers
- **Historical continuity** - Links to typewriter art, teleprinter art
- **Digital vernacular** - Folk art of the internet age
